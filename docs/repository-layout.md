# Repository Layout

## Purpose

This document defines the intended top-level structure of the bonsai repository.

Its purpose is to keep the project organized, make ownership clear, and ensure the repository can grow without frequent restructuring. The layout should reflect subsystem boundaries and support a professional development workflow from the beginning.

bonsai/
